# Forrest Research Foundation research profile

This is a starter personal website for a Forrest Research Foundation Scholar or Fellow. Use it to create a simple public research profile with VS Code, GitHub, and GitHub Copilot.

## Start here

1. Replace `[Your name]` with your name.
2. Replace `[Scholar / Fellow]` with your current appointment.
3. Replace the university placeholder with Curtin, Murdoch, UWA, ECU, or Notre Dame.
4. Add a public-safe description of your research, interests, and links.
5. Preview the site locally, then commit and push your changes.

## Local preview

On macOS, Linux, Git Bash, and most Codespaces:

```bash
python3 -m http.server 8000
```

On Windows PowerShell:

```powershell
py -m http.server 8000
```

Open <http://localhost:8000> in a browser. If Python is unavailable, use VS Code Live Preview or GitHub Codespaces.

## GitHub Pages

The repository includes a GitHub Actions workflow for Pages. In the repository settings, choose **Pages**, then set the source to **GitHub Actions**. A user site repository should be named:

```text
YOUR-USERNAME.github.io
```

## Research and privacy

Publish only information that is safe to make public. Do not paste unpublished thesis chapters, participant data, embargoed results, API keys, passwords, or confidential supervision/review material into Copilot or this repository.
